DataRecord

public protocol DataRecord : NSFetchRequestResult, CoreModel

Represents a uniquely identifiable Core Data record as returned by a fetch request. In most cases this will be an NSManagedObject. The important part is that it be uniquely identifiable.

  • uidKey Default implementation

    The name of the attribute (database column) that represents the unique identifier field. For each custom NSManagedObject subclass you would override this to return the name of the attribute to use. Defaults to "uid".
