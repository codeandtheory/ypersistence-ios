extension DispatchQueue
  • executeOnMain(execute:)

    Executes the block on the main thread. If called from the main thread, the block will be executed immediately and synchronously. If called from a background thread, the block will be executed asynchronously on the main thread. This method is intended to avoid unnecessary calls to DispatchQueue.main.async but you wish to guarantee that the code is run on the main thread.
